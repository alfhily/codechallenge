## Requirements

Ensure you have the following installed on your system:

PHP 8.2 or higher

Composer

Laravel 10

MySQL

Node.js & npm 

XAMPP

PhpSpreadsheet


## Install Dependencies

composer install

npm install


## Configure Environment Variables

Update database details in the .env file:

DB_CONNECTION=mysql

DB_HOST=127.0.0.1

DB_PORT=3306

DB_DATABASE=student_db

DB_USERNAME=root

DB_PASSWORD=
